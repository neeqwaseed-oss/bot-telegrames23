# TCGIS - Telegram Country Group Indexing System 🌍🤖

نظام متكامل لفهرسة مجموعات تيليجرام العامة وتصنيفها حسب الدول مع واجهة بوت احترافية.

## 🚀 المميزات
- 🔍 **بحث متقدم**: بحث سريع في المجموعات باستخدام Elasticsearch.
- 🌐 **تصفح جيو-جرافي**: عرض المجموعات حسب الدولة (يدعم كافة الدول العربية).
- 📊 **إحصائيات**: متابعة نمو المجموعات وأعداد الأعضاء.
- 🤖 **واجهة بوت ذكية**: أزرار شفافة (Inline) وبحث نصي مباشر.
- 🐳 **دعم Docker**: تشغيل النظام بالكامل بلمسة واحدة.

## 🛠 المتطلبات التقنية
- Docker & Docker Compose
- Telegram Bot Token (من @BotFather)

## 📦 خطوات التشغيل على السيرفر (Deployment)

### 1. سحب المشروع
```bash
git clone https://github.com/neeqwaseed-oss/bot-telegrames23.git
cd bot-telegrames23
```

### 2. إعداد المتغيرات البيئية
قم بإنشاء ملف `.env` داخل مجلد `tcgis`:
```bash
cp tcgis/.env.example tcgis/.env
```
قم بتعديل الملف ووضع قيمك الخاصة:
- `BOT_TOKEN`: توكن البوت الخاص بك.
- `ENV`: اضبطه على `production`.
- `WEBHOOK_URL`: رابط السيرفر الخاص بك (يجب أن يبدأ بـ https).

### 3. التشغيل باستخدام Docker
```bash
cd tcgis
docker-compose up -d --build
```

## 📂 هيكلية المشروع
- `api_gateway`: بوابة الربط البرمجية (FastAPI).
- `bot_engine`: المحرك الرئيسي لبوت تيليجرام (Aiogram 3).
- `scraper_engine`: نظام جمع البيانات من المصادر المختلفة.
- `data_processor`: معالجة البيانات وتصنيفها باستخدام الذكاء الاصطناعي.
- `shared`: المكتبات والنماذج المشتركة بين الخدمات.

## 📝 ملاحظات
- النظام مبرمج للتحول التلقائي بين وضع **التطوير** (Polling) ووضع **الإنتاج** (Webhook).
- في وضع التطوير، يتخطى البوت أخطاء الاتصال بالخدمات ليعرض نتائج تجريبية (Demo Mode) لسهولة الاختبار.
